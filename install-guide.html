📱 手机安装指南

把念念日历装到手机桌面，像原生 App 一样使用

🚀 快速安装（推荐）
📦 打包成原生App

🚀 PWA 安装到手机桌面

PWA（渐进式 Web 应用）可以直接安装到手机桌面，无需应用商店，无需下载安装包。
安装后像原生 App 一样全屏运行，支持离线使用和消息通知。

✅无需应用商店
✅免费安装
✅支持离线
✅支持通知

🤖 安卓手机安装

方法一：Chrome 浏览器（推荐）

1
用 Chrome 浏览器打开念念日历网址
2
点击浏览器右上角 ⋮（三个点菜单）
3
找到 "添加到主屏幕" 或 "安装应用"
4
点击确认，桌面就会出现 念念日历图标
5
点击图标即可全屏使用，和原生 App 一样
💡 部分安卓手机（华为、小米、OPPO）自带浏览器也支持，步骤类似：菜单 → 添加到桌面/安装应用

方法二：Edge 浏览器

1
用 Edge 打开网址 → 菜单 ⋮ → 添加到手机 → 安装

🍎 iPhone 安装

⚠️ iPhone 必须用 Safari 浏览器打开，Chrome/微信内置浏览器不支持安装 PWA
1
用 Safari 浏览器打开念念日历网址
2
点击底部 分享按钮（方框+向上箭头 ↑）
3
在弹出的菜单中找到 "添加到主屏幕"
4
点击右上角 "添加"
5
桌面出现念念日历图标，点击即可全屏使用
💡 iOS 上的 PWA 通知功能需要 iOS 16.4+ 系统，且需要从桌面图标打开（不是 Safari 内）

🔔 开启通知提醒

安装到桌面后，还需要开启通知权限，才能在纪念日前一天收到提醒

安卓

1
从桌面打开念念日历 → 进入 "我的" 页面
2
点击 "通知提醒" 开关 → 浏览器会弹出权限请求
3
选择 "允许" → 完成！

iPhone（iOS 16.4+）

1
从 桌面图标打开（不是 Safari）→ 进入"我的"
2
点击 "通知提醒" → 系统弹出权限请求 → 允许
💡 iPhone 需要从桌面图标打开才支持通知，Safari 内打开不支持

📋 PWA 功能清单

  • 全屏运行，隐藏浏览器地址栏
  • 桌面图标，点击直接打开
  • 离线使用，没有网络也能查看已保存的事件
  • 消息通知，纪念日前一天弹窗提醒
  • 数据存储在手机本地，隐私安全
  • 支持安卓和 iPhone 跨平台使用

📦 打包成原生 App（APK/IPA）

如果需要生成真正的安装包（.apk 安卓 / .ipa 苹果），可以使用以下工具将 PWA 包装成原生应用

方案一：PWABuilder（最简单）

1
打开 https://www.pwabuilder.com
2
输入念念日历的网址 → 点击 Start
3
系统会自动分析 → 点击 Package For Stores
4
选择 Android → 下载生成的 .apk 文件
5
将 .apk 传到手机安装即可
💡 PWABuilder 生成安卓 APK 是免费的，苹果 IPA 需要苹果开发者账号（$99/年）

方案二：Capacitor（代码打包）

1
安装 Node.js 和 Capacitor
2
运行命令：
npm install @capacitor/core @capacitor/cli
npx cap init 念念日历 com.memoday.app
npx cap add android
npx cap add ios
npx cap open android
3
在 Android Studio 中点击 Build → Build APK
4
生成的 APK 在 app/build/outputs/apk/ 目录

方案三：Bubblewrap（Google 官方工具）

1
npm install -g @bubblewrap/cli
bubblewrap init --manifest=https://你的网址/manifest.json
bubblewrap build
2
生成的 APK 直接可以安装到安卓手机
⚠️ 苹果 IPA 打包必须有苹果开发者账号（$99/年），且需要 Mac 电脑和 Xcode。安卓 APK 打包完全免费。

📊 两种方案对比

🚀 PWA 安装
✅ 免费、无需打包
✅ 安卓+iPhone 都支持
✅ 更新即时（改代码自动生效）
❌ 需要用浏览器打开再安装
❌ iPhone 通知需 iOS 16.4+
❌ 不能上架应用商店
📦 原生打包
✅ 生成真正的 .apk/.ipa
✅ 可以上架应用商店
✅ 通知功能更稳定
❌ 苹果需开发者账号 $99/年
❌ 打包需要开发环境
❌ 更新需重新打包发布
💡 推荐先用 PWA 安装，日常使用完全够用。如果需要上架应用商店或需要更强的通知能力，再考虑打包成原生 App。
